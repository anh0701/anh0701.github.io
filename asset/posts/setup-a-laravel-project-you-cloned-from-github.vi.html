Cách setup project Laravel clone từ GitHub

Bước 1: Cài đặt Composer Dependencies

composer install

Bước 2: Tạo 1 file .env

cp .env.example .env

Bước 3: Tạo key mã hóa cho ứng dụng

php artisan key:generate

Bước 4: Trong file .env, bạn cần thêm thông tin kết nối cơ sở dữ liệu để Laravel có thể truy cập được. Trong file .env, điền các thông tin DB_CONNECTION, DB_HOST, DB_PORT, DB_DATABASE, DB_USERNAME, DB_PASSWORD.

Bước 5: Chạy migrate cho cơ sở dữ liệu

php artisan migrate

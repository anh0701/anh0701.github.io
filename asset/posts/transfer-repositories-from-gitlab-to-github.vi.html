Chuyển repository từ GitLab sang GitHub có thể trông như một nhiệm vụ khó khăn, nhưng với các bước đúng đắn thì thực ra khá đơn giản. Hướng dẫn này sẽ đưa bạn qua toàn bộ quá trình di chuyển dự án, đồng thời giữ nguyên toàn bộ lịch sử commit, các branch và tag, để đảm bảo việc chuyển đổi giữa hai nền tảng diễn ra suôn sẻ.

Bước 1: Tạo 1 repo trống trên GitHub

Bước 2: Mở repo GitLab, chạy lệnh cmd sau:

git remote add github URL-repo-github;
git push --mirror github

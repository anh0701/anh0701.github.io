Cách cài đặt Spring boot

1. Windows

Bước 1: Cài đặt Scoop

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
Invoke-RestMethod -Uri https://get.scoop.sh | Invoke-Expression

Bước 2: Cài đặt Java

scoop bucket add java extras main
scoop install java/openjdk21 extras/springboot main/maven main/gradle

Bước 3: Cài đặt IntelliJ IDEA Ultimate

scoop install extras/idea-ultimate

2. Linux

Bước 1: Cài đặt zip, unzip, curl:

 sudo apt install unzip zip curl

Bước 2: Cài đặt sdkman: 

curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"
sdk version

Bước 3: Cài đặt gradle, maven, ... bằng sdkman

sdk install java 21.0.8-tem
sdk install gradle
sdk install maven
sdk install springboot
